## Build

- `git clone https://github.com/campos20/mega-sena-gerador.git`
- `cd mega-sena-gerador`
- `yarn install`
- `yarn start`

Abra [http://localhost:3000](http://localhost:3000) para visualizar.

## Tecnologias

- React
- React router
