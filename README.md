## Build

- `git clone https://github.com/campos20/mega-sena-gerador.git`
- `cd mega-sena-gerador`
- `npm install`
- `npm run dev`

Abra [http://localhost:5173](http://localhost:5173) para visualizar.

## Tecnologias

- React
- React router
- Vite
